# User Commands

- `/start` - главное меню.
- `/menu` - открыть главное меню и сбросить текущий сценарий.
- `/share` - ссылка на бота для открытия с другого Telegram-аккаунта. Данные аккаунтов изолированы друг от друга.
- `/help` - краткая справка.
- `/version` - текущая бета-версия бота и подсказка, где посмотреть подробности.
- `/advisor` - открыть раздел `Птицевод`.
- `/my_id` - показать ваш Telegram ID для настройки `ADMIN_IDS`.
- В основных Telegram-разделах есть кнопка `FAQ` с короткой справкой по текущему экрану.

## Инкубация

- `/new` - добавить партию инкубации.
- `/batches` - активные партии.
- `/today` - план на сегодня.
- `/history` - завершенные партии.
- `/stats` - статистика вывода.
- `/profiles` - режимы инкубации.
- `/calendar` - календарь работ.
- `/care` - уход после вывода.
- `/edit ID поле значение` - быстро изменить партию.
- `/reopen ID` - вернуть завершенную партию в активные.

## Корма, склад, поголовье и стада

- `/feed` - добавить корм или ингредиент на склад.
- В разделе `Корма -> Склад` можно добавить покупку, сделать замес смеси, посмотреть расчетные остатки, историю и назначить рацион поголовью.
- При замесе бот показывает формулу на один базовый цикл в объемных частях, дает чеклист добавленных ингредиентов без дублирования сообщений и после кнопки `Замес готов, обновить склад` списывает потраченные ингредиенты и добавляет готовую смесь.
- Остаток готовой смеси уменьшается расчетно по активным рационам без ежедневных дублей операций.
- Раздел `Корма -> Смесь` показывает, сколько полных замесов можно сделать из текущих складских остатков, и предлагает быстрые кнопки расчета. Если выбрано несколько замесов, они идут как повторы: `1 из N`, `2 из N` и так далее.
- Раздел `Корма -> Поголовье и стада -> Поголовье` - отдельные группы птиц: несушки, петухи, цыплята.
- Раздел `Корма -> Поголовье и стада -> Стада` - набор групп поголовья, которые едят одну готовую смесь.
- В карточке стада можно изменить состав и назначить готовую смесь. Сейчас назначение смеси ставится как 100% рациона.
- Раздел `Корма -> Расчеты` показывает дневной расход по стадам, запас готовой смеси, сколько еще можно произвести из складских ингредиентов и ориентировочные даты закупок.
- Для цыплят указывается дата вывода и примерная дата подсадки в основной курятник; расход считается по возрасту с запасом 10%.
- В карточке корма: пополнить, списать, редактировать, сменить группу, посмотреть историю, архивировать.

## Яйца

- Раздел `Яйца` открывается из главного меню.
- Главная страница `Яйца` показывает погоду в формате `день/ночь + завтра`; если кеш устарел, бот открывает экран сразу и обновляет погоду в фоне.
- `Добавить яйца` - выбрать `Сегодня` или `Вчера` и внести количество собранных яиц.
- `Расчеты` - показать сбор за сегодня, среднее за 7/30 дней, прогноз на неделю, погодную поправку и расчет на одну активную несушку.
- `История` - последние 14 дней с нулевыми днями.
- `Не несутся` - временно исключить несушек из расчета: наседка, уход за цыплятами, линька, болезнь или другая причина.
- `Город и погода` - сохранить город, посмотреть последнюю сохраненную погоду и при необходимости обновить прогноз через Open-Meteo.

## Птицевод

- Раздел `Птицевод` открывается из главного меню или командой `/advisor`.
- `План на сегодня` - единый список действий по воде, корму, яйцам, смеси и активным инкубационным партиям.
- `Корма и замес` - управленческий вывод по стадам, расходу, запасу готовой смеси и недостающим ингредиентам.
- `Когда замес` - расчет срочности следующего замеса по остатку готовой смеси и текущему расходу.
- `Мало яиц` - анализ 7/30 дней, временно не несущихся кур, погодной подсказки и базовых проверок условий.
- `Инкубация сегодня` - краткие задачи по активным партиям.
- `Проблема с птицей` - безопасный сценарий: красные флаги, изоляция, вода, тепло/сухость/вентиляция и рекомендация обратиться к ветеринару. Бот не ставит диагнозы и не назначает лекарства.

## Настройки

- `/settings` - настройки уведомлений, хозяйства и экран `О боте` с версией, GitHub и changelog.
- `/remind` - текущий статус напоминаний.
- `/remind 09:00` - включить или изменить время ежедневных напоминаний.
- `/remind off` - выключить ежедневные напоминания.
- `/timezone Europe/Moscow` - изменить часовой пояс.
- `/farm Мое хозяйство` - задать название хозяйства.
- `/units metric` - метрические единицы.
- `/disclaimer` - справочный ветеринарный дисклеймер.
- `/cancel` - отменить текущий сценарий и вернуться в меню.

Время из `/remind 09:00` совпадает со временем в `/settings` и применяется с учетом пользовательского часового пояса из `/timezone`.
В `Настройки -> Разделы и уведомления` можно выключить инкубацию, корма, яйца, птицевода, уход после вывода и сервисные сообщения. Выключенный раздел пропадает из главного меню.

Дополнительно worker один раз в день после 12:00 по пользовательскому часовому поясу отправляет ежедневную сводку хозяйства. Эту же сводку можно запросить вручную кнопкой `📊 Посмотреть сводку` в главном меню. В ней есть напоминания собрать яйца, проверить и заменить воду, дать корм, статистика яиц, дата последнего замеса, остаток готовой смеси и короткий блок `Совет птицевода`, если есть полезное действие. Если смесь уже закончилась по расчету, сводка показывает дату окончания и сколько дней назад это было. Если смеси осталось на 2 дня или меньше, сводка показывает критическое предупреждение и подсказку по замесу. Под сводкой есть кнопки `Добавить яйца`, `Переход в раздел корма` и `Выйти в меню`.

Админские команды доступны только `ADMIN_IDS`:

- `/admin` - статистика.
- `/admin_export` - CSV с агрегированной аналитикой.
- `/admin_broadcast текст` - сервисная рассылка с подтверждением.
