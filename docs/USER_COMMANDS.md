# User Commands

- `/start` - главное меню.
- `/share` - ссылка на бота для открытия с другого Telegram-аккаунта. Данные аккаунтов изолированы друг от друга.
- `/help` - краткая справка.
- `/version` - текущая бета-версия бота.
- `/new` - добавить партию инкубации.
- `/batches` - активные партии.
- `/today` - план на сегодня.
- `/history` - завершенные партии.
- `/stats` - статистика вывода.
- `/profiles` - режимы инкубации.
- `/calendar` - календарь работ.
- `/care` - уход после вывода.
- `/feed` - добавить корм или ингредиент на склад.
- В разделе `Корма -> Склад` можно добавить покупку, сделать замес смеси, посмотреть расчетные остатки, историю и назначить рацион поголовью.
- При замесе ингредиенты списываются со склада, а готовая смесь добавляется отдельной складской позицией.
- Остаток готовой смеси уменьшается расчетно по активным рационам без ежедневных дублей операций.
- Раздел `Корма -> Смесь` показывает, сколько полных замесов можно сделать из текущих складских остатков, и предлагает быстрые кнопки создания замеса.
- Раздел `Корма -> Поголовье` - создать основное стадо или цыплят.
- Для цыплят указывается дата вывода и примерная дата подсадки в основной курятник; расход считается по возрасту с запасом 10%.
- В карточке корма: пополнить, списать, редактировать, сменить группу, посмотреть историю, архивировать.
- `/settings` - настройки уведомлений и хозяйства.
- `/remind` - текущий статус напоминаний.
- `/remind 09:00` - включить или изменить время ежедневных напоминаний.
- `/remind off` - выключить ежедневные напоминания.
- `/timezone Europe/Moscow` - изменить часовой пояс.
- `/farm Мое хозяйство` - задать название хозяйства.
- `/units metric` - метрические единицы.
- `/disclaimer` - справочный ветеринарный дисклеймер.
- `/reopen ID` - вернуть завершенную партию в активные.
- `/cancel` - отменить текущий сценарий и вернуться в меню.

Время из `/remind 09:00` совпадает со временем в `/settings` и применяется с учетом пользовательского часового пояса из `/timezone`.

Админские команды доступны только `ADMIN_IDS`:

- `/admin` - статистика.
- `/admin_export` - CSV с агрегированной аналитикой.
- `/admin_broadcast текст` - сервисная рассылка с подтверждением.
